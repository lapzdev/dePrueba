Etiquetas Semánticas Estructurales

Esta etiqueta contenedora no tiene semántica
header- Cabecera de un sitio web o sección
main-Define la Sección Principal del Documento. Solo puede existir una etiqueta main por documento
footer-pie de página de un sitio web o de una sección
nav - Representa una navegación
article - Representa una sección de autocontenido( que por si sola se explica)
aside - Representa un contenido complementario o secundario
section - Representa una sección de contenido
address - Representa una información de contacto

Etiquetas de Bloque vs Etiquetas de Línea

La etiqueta de bloque por excelencia es div; una etiqueta de bloque es aquella que genera saltos de línea con sus hermanas, además de ocupar todo el ancho del img/Perfil.pngelemento
La etiqueta de línea por excelencia es span. una etiqueta de línea sólo ocupa el espacio necesario que tiene su contenido. hola soy otra span

Lista de Definiciones

HTML
Lenguaje de Marcado de Hypertexto(Hyper Text Markup Language en Inglés): Es un lenguaje de marcado que define el contenido de la red.
CSS
Hojas de Estilos en Cascada(Cascade Style Sheet en Inglés):Es un lenguaje de definición que da estilo al HTML.
JavaScript
Es el lenguaje de programación de la Web.
